教程设计基本原则

  • 尽可能讲解透彻，提供入门级案例

  • 尽可能使用通用的数据与模块，避免相似案例下载不同数据集与模块

  • 尽可能适配更多的电脑，包含win、linux

  • 尽可能纯本地运行，避免使用云服务

  • 尽可能使用稳定的版本，保证代码长期有效

Python建议版本：3.11
